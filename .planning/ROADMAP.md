# ROADMAP.md

## Milestone 1: FlashMind MVP

### Phase 1: Project Scaffold & Core Layer
- Initialize Flutter project structure.
- Implement `core/` (Constants, Validators).
- Implement `data/` (Model, Storage Service, Repository).

### Phase 2: App Config & Styling
- Configure `app/theme/` (Poppins, Colors).
- Setup `app/routes/` and `app/bindings/`.
- Initialize `FlashMindApp` in `main.dart`.

### Phase 3: Home Feature (The Core Experience)
- Implement `HomeController` with navigation logic.
- Build `FlipCard` widget (3D transformation).
- Build `NavButtons` and `HomeScreen` UI.

### Phase 4: Manage Feature (The CRUD Layer)
- Implement `ManageController` with search/filter logic.
- Build `CardFormSheet` (Add/Edit) and `CardListItem` widgets.
- Build `ManageScreen` with Swipe-to-Delete.

### Phase 5: Integration & Verification
- Finalize cross-controller synchronization.
- Perform UI/UX polish and animation tuning.
- Complete QA/Testing audit.
